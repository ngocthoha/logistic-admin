# CREATE PROJECT VUEJS

## Prerequisites

- NodeJS.

- yarn.

- Git

## Start

### Installation

```
$ yarn global add @vue/cli
$ vue create my-project
$ cd my-project
```
### Project setup

```
$ yarn install
```

### Complies and hot-reloads for development

```
$ yarn serve
```

### Complies and minifies for production

```
$ yarn build
```
### Lints and fixes files

```
$ yarn lint
```
### Add Sass

```
$ yarn add sass-loader
```
